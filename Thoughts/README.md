# Thoughts

A monthly rolling log of raw, unpolished thoughts — signals, reactions, fragments, half-formed observations. Captured fast, developed slowly.

## Format

Each month is one file: `YYYY-MM.md`

Entries are dated blocks:

```
### 2026-03-12
Your thought here. Can be one line or a paragraph.

> **2026-03-14**: Annotation or follow-up — adds context without changing the original.
```

## Rules

- **Never edit original captures.** If you want to add to a thought, append an annotation block (`> **date**: ...`) below it.
- **No classification required.** Themes, connections, and promotions happen later via `/ideas-review` and `/develop`.
- One thought per `### date` block. Multiple thoughts on the same day get separate `### date` blocks.

## Lifecycle

1. Capture with `/thought` — goes here
2. Annotate with `/thought annotate` — appends context to an existing entry
3. Promote with `/idea` — when a thought earns its own file and lifecycle
4. Find patterns with `/ideas-review` — surfaces promotable thoughts and themes
