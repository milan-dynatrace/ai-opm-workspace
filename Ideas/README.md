# Ideas

Developed ideas worth tracking — promoted from the Thoughts log or captured directly when a concept is already formed enough to deserve its own lifecycle.

## What belongs here

A thought graduates to an idea when it has:
- Enough substance to stress-test
- A clear "Why This Matters"
- Something to do next (even if just "think more about this")

If a thought doesn't meet that bar yet, it stays in `Thoughts/`.

## File naming

`YYYY-MM-DD - [Title].md`

## Lifecycle statuses

- `seed` — just promoted, not yet developed
- `growing` — actively being developed (via `/develop`)
- `ready` — strong enough to synthesize into a plan/narrative/decision
- `archived` — no longer relevant, but preserved
- `shipped` — turned into a real output or decision
