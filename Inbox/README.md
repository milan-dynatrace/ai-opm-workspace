# Inbox

Drop anything unstructured here — no classification required at capture time.

## What belongs here

- Braindumps and free-form notes
- Pasted Slack threads or email snippets
- Voice-to-text exports
- Screenshots of content you want to process
- Quick captures that don't have a home yet

## What does NOT belong here

- Meeting notes → `Meetings/` (processed with `/meeting`)
- Already-structured decisions or loose notes → `Loose Notes/Work/`

## How it works

1. Drop a `.md` file here with any content — name it anything
2. Run `/process-inbox` to review, classify, and route each file
3. The original file is moved to `Archive/Notes Archive/` after processing

## Naming suggestion (not required)

`YYYY-MM-DD - anything you want.md` — the date helps with context when reviewing later.
